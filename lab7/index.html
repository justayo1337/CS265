Introduction

Here are some suggestions for functions to write, by way of getting practice with C, with pointers, and with linked lists and arrays.

You might find the Tiny C Compiler helpful. It is installed on tux. Interestingly, you can simply run your program from source, helpful for recompiling/developing/debugging:

$ tcc -run sample.c arg1 arg2

(Note that the -run is not a POSIX long option.)

I've not played around w/multiple file-compiles and the -run option yet. I'll get there.

Or, simply use gcc. This always works:

$ gcc sample.c -o sample
$ ./sample arg1 arg2

Strings

Consider this alternative declaration:

int myStrlen( char s[] ) ;

Consider this alternative declaration:

void myStrcpy( char t[], char s[] ) ;

Remember, all of the C library functions are described in section 3 of the manpages, so:

$ man -s3 printf

might be helpful.

Note that all lower-case letters are contiguous, as are the upper-case. Remember that characters are really just 1-byte integers. If we find the difference, we can use that to convert lower-case to upper-. 

int diff = 'a' - 'A' ;
c = 'k' ;
if( 'a'<=c && c<='z' )   // then it's lower-case
   c -= diff ;
