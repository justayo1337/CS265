The Exercises

  1. Look at the files for this lab. This stack is very much like the one we looked at in this week's activity, with one notable change. The underlying array is allocated off the heap, so that it can grow; 

    struct stack-arr 
{
   ELEM *arr ;
   size_t size ;
   size_t cap ;
}

    I replaced the init function with a factory function, mkStack, which returns a pointer to a new (and initialised) stack from the heap. The client should start with a pointer, and call this method to get a new stack. I moved the struct itself into the implementation file.

    Your job is to define the stackResize function. See stack.c .
