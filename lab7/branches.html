The Exercises

    Boolean Expressions

  1. Consider the following initialisations: 
    int i = 5 ,
    j = 12 ,
    k = -13 ,
    z = 0 ;
    1. j < i
    2. ! j < i
    3. (! j) < i
    4. i + j
    5. !i + j
    6. !!i + !j
    7. j && k
    8. i && j || z
    9. i && ( j || z )

    2. If Statements

  2. A leap year must be divisible by 4. If the year is divisible by 100 it is not a leap year, unless it is also divisble by 400.

    Write a program called leap.c which prompts the user for a year (as an int), prints YES if the input year is a leap year, and NO otherwise.
